## .NET User Secrets

In Rider: Rechtsklick auf das Testprojekt --> Tools --> .NET User Secrets. 

### Benötigte Secrets
- ServerUrl 
  - URL zum FE2 Server (endet nicht mit "/")
- Username 
  - FE2-Benutzername
- Password
  - FE2-Passwort
- TestCalendarName
  - Bezeichnung des Kalenders, in dem Tests ausgeführt werden.
